## Project Overview
This project is part of the Data Analyst Internship Task 2, where the objective is to create meaningful visualizations and present a clear data story using Power BI.  
The dataset used is **Superstore Sales Data**, which includes sales, profit, quantity, category, state, and region information.

## Task Objective
- Create a set of visualizations using the correct chart types  
- Add context, legends, and insights to each visual  
- Build a visual story that explains the business performance  
- Avoid clutter, highlight key takeaways, and follow best design practices  
- Export the dashboard screenshots or PBIX/PBIT file for submission  

## What I Did in This Task

1. **Imported and cleaned the Superstore dataset** using Power BI.
2. Created **KPI Cards** showing:
   - Total Sales  
   - Total Profit  
   - Total Quantity Sold  
3. Built multiple visualizations:
   - **Sales by Category** (Column Chart)
   - **Sales by Sub-Category** (Horizontal Bar Chart)
   - **Profit by Region** (Column Chart)
   - **Monthly Sales Trend** (Line Chart)
   - **Sales by State** (Filled Map)
4. Added **legends, labels, colors, and tooltips** for clarity.
5. Designed a clean, easy-to-read layout with minimal clutter.
6. Created a final **Insights & Summary** section explaining the key findings.
7. Saved the dashboard as **Power BI Template (.pbit)** for submission.
8. Prepared this README.md to document the entire process.

## Visuals Included

- Category-level and Sub-category-level Sales Analysis

## Key Insights from the Dashboard
- **Technology** is the highest revenue-generating category.  
- **Phones** and **Chairs** appear as top-performing sub-categories.  
- **West and East regions** generate the most profit.  
- Low performance observed in **South and Central regions**.  
- Sales peak during **Octoberâ€“December**, indicating seasonal trends.  
- Several states have high sales potential but underperform currently.
